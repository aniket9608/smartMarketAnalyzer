# Smart Market Data Analyzer

A Python project that scrapes product data from a website, stores it in CSV format,
performs price analysis using Pandas, and visualizes results using Matplotlib.

## Features
- Web scraping using BeautifulSoup
- Object-Oriented Programming (OOP)
- Data Analysis with Pandas
- Data Visualization with Matplotlib
- Exception Handling

## How to Run
python3 marketanalyzer.py
